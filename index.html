Add Workout Tracker to your Home Screen

  1. 1Open this website in Safari.
  2. 2Tap the Share button at the bottom of Safari.
  3. 3Scroll down and choose Add to Home Screen.
  4. 4Tap Add. The tracker will open like an app and continue working offline.

The files must be hosted on an HTTPS website. Opening the HTML file directly from the iPhone Files app cannot install the offline app.
